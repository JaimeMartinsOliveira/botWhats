## Projetos Principais

### LlamaScrapeAsync
- **Descrição:** Prova de conceito que integra um modelo de linguagem local (Hugging Face) com automação de navegador (Playwright) e interface com NiceGUI. O objetivo é permitir que um agente de linguagem natural controle um navegador para buscar informações e interagir com páginas web.
- **Tecnologias:** Python, Pandas, transformers, LangChain, Playwright, NiceGUI, Google FLAN-T5 Large.
- **Link:** https://github.com/JaimeMartinsOliveira/LlamaScrapeAsync

### Easy-Finance: Finanças Inteligentes para Autônomos
- **Descrição:** App de finanças pessoais para autônomos. Back-end desenvolvido com Python (FastAPI) e front-end com Ionic/React. Unifica transações bancárias (via API Open Finance da Belvo) com registros manuais. O diferencial é um assistente com IA que gera análises e metas financeiras.
- **Tecnologias:** Python, FastAPI, PostgreSQL, Docker, N8N, Belvo Api, React, Ionic, JWT.

### Documentos-Infraestrutura-NLP
- **Descrição:** Assistente inteligente treinado para reconhecer e classificar automaticamente descrições de atividades em documentos e relatórios de gestão de infraestrutura pública. Utiliza um modelo de Machine Learning (Random Forest) com vetores TF-IDF e possui uma interface interativa com Gradio.
- **Tecnologias:** Python, NLTK, Pandas, NLP, Scikit-learn, Streamlit, transformers.
- **Link:** https://github.com/JaimeMartinsOliveira/Documentos-Infraestrutura-NLP

### infra-doc-classifier
- **Descrição:** Classificador de documentos técnicos de infraestrutura utilizando NLP com Transformers. O projeto visa classificar automaticamente documentos por setores (energia, transporte, saneamento), ideal para organizações que lidam com grandes volumes de textos técnicos.
- **Tecnologias:** Python, NLTK, Pandas, NLP, Scikit-learn, Streamlit, transformers, NumPy.
- **Link:** https://github.com/JaimeMartinsOliveira/infra-doc-classifier
